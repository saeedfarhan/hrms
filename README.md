# HRMS Lite 🚀

## Project Overview

**HRMS Lite** is a lightweight, production-ready **Human Resource Management System** designed for small teams and startups. It provides essential HR functionality with a modern, intuitive interface:

- **Employee Management**: Add, view, edit, and delete employee records
- **Attendance Tracking**: Mark daily Present/Absent status with history
- **Single Page Application**: Tab-based navigation (no routing complexity)

Built for **6-8 hour development sprints** with focus on **clean code, scalability, and developer experience**.

## Tech Stack
- Backend: FastAPI (Python 3)
- Database: MySQL (with SQLAlchemy ORM)
- Frontend: React
- Styling: Tailwind CSS 3.4 (Utility-first)
- HTTP: Native Fetch API (No Axios)
- Validation: Pydantic (Strict server-side)


### Steps to run locally
-  Start FastAPI server (fastapi dev)
-  Start React (npm run start)
- Configure db url in backend/app/db.py
